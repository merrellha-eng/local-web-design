# LOCAL WEB DESIGN

`LOCAL WEB DESIGN` のランディングページ、お問い合わせページ、完了案内ページを含む静的サイトです。

## ファイル構成

- `index.html`
- `contact.html`
- `thanks.html`
- `css/style.css`
- `js/main.js`
- `images/`
- `.nojekyll`

## ローカル確認方法

`index.html` をブラウザで開いて表示を確認してください。お問い合わせフォームの送信ボタンは、入力内容を送信せず `thanks.html` へ遷移します。

## GitHub Pagesでの公開手順

1. このフォルダ内のファイル一式を手動でGitHubリポジトリへ配置します。
2. GitHubリポジトリの Settings から Pages の公開元ブランチとフォルダを設定します。
3. 公開URLにアクセスし、トップページ、問い合わせページ、完了ページ、画像、CSS、JavaScriptが正しく表示されることを確認します。

## お問い合わせフォームに関する重要事項

現在の `contact.html` は表示確認用のダミーフォームです。送信ボタンを押すと、入力内容を送信・保存・通知せず `thanks.html` へ遷移するだけです。

現在の状態では問い合わせの受信、保存、メール通知は一切できません。運用開始前に問い合わせの受信方法を別途決定し、実際の送信処理を実装してください。
